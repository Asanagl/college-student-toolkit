# Checklist

## 项目骨架
- [ ] Electron + React + TypeScript 项目骨架搭建完成，`npm run dev` 可启动开发模式
- [ ] 应用主界面有左侧导航栏，五个板块入口可点击切换并显示对应占位页面
- [ ] 主进程服务层骨架已定义（DataService、NotificationService、ScheduleService、TimetableService、ExamService、LabService、GradeService、UpdateService）
- [ ] contextBridge 已暴露安全的 window.api 接口，IPC 通道命名规范统一

## 数据存储与基础设施
- [ ] SQLite 数据库 schema 覆盖所有数据类型（日程、课程、自定义时间、考试、实验、成绩、GPA 公式）
- [ ] 所有数据存储在 Electron app.getPath('userData')
- [ ] DataService 通过 IPC 暴露 CRUD 接口给渲染进程
- [ ] 数据导出为单一 ZIP 文件（archiver）
- [ ] 数据导入支持格式校验和覆盖确认（dialog.showOpenDialog）
- [ ] 导入后所有板块数据正确刷新
- [ ] Windows 11 通知通过 Electron Notification API 推送，可在通知中心查看
- [ ] 通知免打扰时段生效，免打扰结束后汇总推送

## 日程板块
- [ ] 日历视图支持月份切换、日期选择、日程标记点显示
- [ ] 日程 CRUD 对话框字段完整（标题、开始/结束时间、提前时间、重复规则）
- [ ] 日程提醒在指定时间（日程时间 - 提前时间）触发 Windows 通知
- [ ] 重复规则（每日/每周/每月）正确生成下次触发时间

## 课程表板块
- [ ] 课程表周视图布局参考 Wake Up 课程表样式（7 列 × N 节网格，CSS Grid）
- [ ] 课程卡片按颜色、节次、周次正确渲染
- [ ] 周次切换控件工作正常（上一周/下一周/当前周）
- [ ] 手动添加课程对话框字段完整（含周次、节次、颜色）
- [ ] 自定义时间设置可编辑每节课起止时间、课间休息、大课休息、实验时间
- [ ] 每节课前提醒提前时间可配置
- [ ] 嵌入式浏览器（Electron BrowserWindow）可登录教务系统并导航至课程表页面
- [ ] DOM 解析器至少支持一种主流教务系统（正方/URP/青果），使用 cheerio 解析
- [ ] OCR 识别能对课程表截图进行二次识别（tesseract.js + chi_sim）
- [ ] DOM 与 OCR 冲突时，UI 高亮差异供用户选择
- [ ] 用户确认后课程被批量导入
- [ ] 课前提醒按自定义时间 + 提前时间触发 Windows 通知

## 考试安排板块
- [ ] 考试安排支持手动和自动导入
- [ ] 自动导入复用嵌入式浏览器、DOM 解析、OCR 二次确认流程
- [ ] 考试手动添加对话框字段完整（含座位号、备注）
- [ ] 多档提醒（如提前1周、3天、1天）可同时设置
- [ ] 修改考试时间后所有提醒重新调度
- [ ] 考试列表按日期排序并显示倒计时

## 实验提醒板块
- [ ] 实验提醒支持手动和自动导入
- [ ] 实验手动添加对话框字段完整（含指导教师、注意事项）
- [ ] 实验提醒在用户设定的提前时间触发 Windows 通知
- [ ] 实验列表按日期排序并显示倒计时

## 成绩显示板块
- [ ] 成绩列表按学期分组，支持 Tabs 切换学期
- [ ] 成绩手动添加对话框字段完整（含学期、是否学位课）
- [ ] GPA 公式编辑器支持变量、运算符、Σ 聚合函数
- [ ] 公式语法校验生效（expr-eval），错误时高亮错误位置
- [ ] GPA 计算结果正确（手动验证至少 3 个样例）
- [ ] 本学期 GPA、总体 GPA、学位课 GPA 三类分别展示

- [ ] update-downloaded 事件触发后弹窗提示用户安装

- [ ] 性能测试：本地数据库查询 < 100ms，React 渲染流畅，通知调度延迟 < 1s
